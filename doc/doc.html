/schedules

gestiona los programas del controlador. Las horas se establecen en horario UTC.

/schedules get

get

obtiene todos los programas

get /schedules

obtiene todos los programas

  • Request
  • Response

Headers

  • Authorization: (string)

    Example:

    apikey

HTTP status code 200

Body

Type: application/json

Example:

{
	"schedules": [
		{
			"_id": "schedule01",
			"name": "findes",
			"start": "2007-11-03T13:18:05",
			"end": "2007-11-03T13:19:05",
			"repeat": []
		}
	]
}
